# VN 剧情树与背景项目（结构化版）

## 目录结构
- `index.html`：单页工作台
- `styles.css`：页面样式
- `app.js`：剧情树渲染与导出
- `story_seed.json`：初始节点种子
- `assets/logo.png`：可见 logo
- `preview.png`：可见预览图
- `docs/00_project_overview.md`：项目概览
- `docs/01_story_tree.md`：剧情树主文档
- `docs/02_world_background.md`：世界观/背景
- `docs/03_character_bible.md`：角色圣经
- `docs/04_scene_blueprint.md`：场景蓝图
- `docs/05_branch_rules.md`：分支规则

## 用法
1. 打开 `index.html`
2. 调整项目名和一句话概述
3. 点击导出 JSON 或 Markdown
4. 在 `docs/` 下继续补写细节
