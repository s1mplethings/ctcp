# Run Instructions

1. Open the copied project directory under `01_project/project_dir/`.
2. Start the app with `python app.py`.
3. Visit `http://127.0.0.1:8008`.
4. Upload a CSV file, choose cleanup options, and export the cleaned CSV.

Cold replay entry:
- `python scripts/run_project_web.py --serve`
- `python scripts/run_project_web.py --goal "replay smoke export" --project-name "CSV Cleaner Studio" --out generated_output`
