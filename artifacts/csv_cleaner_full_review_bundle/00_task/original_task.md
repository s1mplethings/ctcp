# 原始任务说明

任务来源：当前对话中的用户请求。

固定实战项目：CSV 数据清洗与导出 Web 工具。

最低必须功能：
- 上传 CSV 文件
- 预览原始表格前几行
- 删除空行
- 删除重复行
- 选择保留列
- 预览清洗后的结果
- 导出清洗后的 CSV
- 页面必须像成品工具页
- 必须有 README / 运行说明
- 必须能打成最终项目包
- 必须走交付闭环
- 必须通过冷复跑验证

额外打包要求：
- 保存任务说明、task/spec/plan、过程记录
- 保存最终高价值截图和 `replayed_screenshot.png`
- 保存最终项目包和项目目录副本
- 保存环境信息清单
- 保存 `support_public_delivery.json`、`replay_report.json`、completion gate、verify/simlab/workflow 输出
- 输出一个带 `INDEX.md` 的总包，方便外部复审
