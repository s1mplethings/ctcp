# Original Project Request

## Project Name
Batch Image Processor

## Project Goal
Build a locally runnable web tool that lets users upload multiple images, perform real image processing, preview the results, download single outputs, and download all processed results as one zip.

## Minimum Required Features
1. Upload multiple `jpg/jpeg/png` images at once.
2. Show the original image list and thumbnail previews.
3. Support at least four processing capabilities:
   - fixed width/height resize
   - proportional resize
   - output format conversion (`png/jpg`)
   - compression quality adjustment
4. Support optional rename rules:
   - keep original filename
   - add prefix
   - add sequence number
5. Preview processed results.
6. Support single-file download.
7. Support one-click zip download for all processed files.
8. Page should look like a finished tool page, not a debug page.
9. Include README / run instructions.
10. Produce a final project package.
11. Enter the delivery closure.
12. Pass cold replay.

## User-Specified Final Outputs
- final project package
- final high-value screenshot
- `support_public_delivery.json`
- `replay_report.json`
- `replayed_screenshot.png`
- README / run instructions
