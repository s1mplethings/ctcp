# AI Doc Scaffold

Use these templates to bootstrap `docs/aidoc/*` in a target project.

## Files
- `context.md`
- `architecture.md`
- `flows.md`
- `prompts.md`
- `decisions.md`
- `checklists.md`

## Usage
- PowerShell: `powershell -ExecutionPolicy Bypass -File scripts/gen_aidoc.ps1 -Target "C:\\path\\to\\project"`
- Bash: `bash scripts/gen_aidoc.sh /path/to/project`
