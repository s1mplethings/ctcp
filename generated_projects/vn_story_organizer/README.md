# VN Story Organizer

A tool for organizing and structuring Visual Novel (VN) storylines.

## Features

- Parse and organize VN script files
- Visualize story branches and character routes
- Track scene dependencies and flow
- Export organized storyline summaries

## Quick Start

`ash
python main.py
`

## Project Structure

`
vn_story_organizer/
  main.py        # Entry point
  README.md      # This file
`

## Usage

Run the main script to start the story organizer:

`ash
python main.py
`

The tool will guide you through:
1. Loading your VN script files
2. Parsing scene and dialogue data
3. Organizing storylines by route/character
4. Generating a structured summary

## Requirements

- Python 3.8+
